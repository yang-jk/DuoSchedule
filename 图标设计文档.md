# 双人课程表 - 图标设计文档

## 设计理念

### 核心元素
图标融合了三个核心元素，体现「双人课程表」的产品特色：

1. **双人元素** 👥
   - 左侧橙色人形：代表 Person A（`#FFB74D`）
   - 右侧蓝色人形：代表 Person B（`#4789FE`）
   - 两个抽象人形并肩站立，体现协作与共享

2. **时间元素** ⏰
   - 时钟外环：圆形时钟轮廓，代表课程时间安排
   - 时钟刻度：四个基本方位刻度（12/3/6/9 点方向）
   - 时间指针：时针和分针，指向当前时间
   - 中心圆点：时钟的轴心

3. **课程元素** 📚
   - 底部书本：抽象的书本形状，代表课程学习
   - 半透明效果：与玻璃拟态风格融合

### 视觉风格：玻璃拟态 (Glassmorphism)

- **半透明层次**：多层半透明元素叠加，创造深度感
- **渐变效果**：蓝色系渐变，使用项目主题色
- **高光效果**：顶部椭圆高光，模拟玻璃反光
- **柔和阴影**：轻微阴影增加立体感
- **磨砂质感**：通过透明度和模糊实现

### 配色方案

使用项目现有的蓝色系配色，保持品牌一致性：

| 颜色 | 色值 | 用途 |
|------|------|------|
| 主蓝色 | `#007AFF` | 时钟外环、时间指针、人形 B |
| 亮蓝色 | `#4789FE` | 渐变辅助色、人形 B 渐变 |
| 橙色 | `#FFB74D` | 人形 A |
| 白色 | `#FFFFFF` | 高光、书本、刻度 |

### 透明度层次

- 背景层：15% 透明度
- 玻璃高光：30-40% 透明度
- 主要元素：70-95% 透明度
- 装饰元素：20-60% 透明度

## 图标文件清单

### 1. 应用桌面图标

#### SVG 源文件
- **文件路径**: `app/src/main/res/drawable/ic_launcher_foreground_new.svg`
- **尺寸**: 108x108dp
- **用途**: 设计源文件，可用于其他平台或重新导出

#### Android VectorDrawable
- **文件路径**: `app/src/main/res/drawable/ic_launcher_foreground.xml`
- **尺寸**: 108x108dp
- **用途**: 
  - 应用主图标（前景）
  - 自适应图标（Android 8.0+）
  - 启动页图标
- **引用位置**:
  - `@mipmap/ic_launcher` (通过 `ic_launcher.xml`)
  - `@mipmap/ic_launcher_round` (通过 `ic_launcher_round.xml`)
  - `@style/Theme.DuoSchedule.Splash` (启动页)

### 2. 通知栏图标

#### SVG 源文件
- **文件路径**: `app/src/main/res/drawable/ic_notification.svg`
- **尺寸**: 24x24dp
- **用途**: 设计源文件

#### Android VectorDrawable
- **文件路径**: `app/src/main/res/drawable/ic_notification.xml`
- **尺寸**: 24x24dp
- **用途**: 
  - 通知栏小图标
  - 状态栏图标
- **引用位置**:
  - `PromotedNotificationBuilder.kt` (所有通知)
  - `CourseNotificationManager.kt` (课程提醒和上课中通知)

### 3. 自适应图标配置

#### 标准图标
- **文件路径**: `app/src/main/res/mipmap-anydpi-v26/ic_launcher.xml`
- **配置**: 
  ```xml
  <background>: @color/primary (蓝色背景)
  <foreground>: @drawable/ic_launcher_foreground
  ```

#### 圆形图标
- **文件路径**: `app/src/main/res/mipmap-anydpi-v26/ic_launcher_round.xml`
- **配置**: 同标准图标
- **用途**: 某些启动器使用圆形图标

## 技术实现

### VectorDrawable 优势

1. **无损缩放**: 矢量图形，任意尺寸都清晰
2. **小体积**: XML 格式，比 PNG 小得多
3. **动态修改**: 可通过代码修改颜色、大小等属性
4. **自适应**: 自动适配不同屏幕密度

### 玻璃拟态实现

通过以下 XML 技术实现玻璃效果：

```xml
<!-- 半透明填充 -->
android:fillColor="#26007AFF"  <!-- 15% 透明度 -->

<!-- 渐变效果 -->
<gradient
    android:startColor="#007AFF"
    android:endColor="#4789FE"
    android:angle="45"/>

<!-- 描边效果 -->
android:strokeColor="#B3FFFFFF"  <!-- 70% 透明度白色 -->
android:strokeWidth="2"
```
